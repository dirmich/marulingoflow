# Lingo-Flow 학습 가이드: 단어와 문장을 오래 기억하는 방법

Lingo-Flow는 단순한 암기를 넘어, 뇌의 기억 원리를 활용한 과학적인 학습 방법을 지향합니다. 단어와 문장을 더 쉽고 오래 기억하기 위한 핵심 전략을 소개합니다.

---

## 1. SRS (Spaced Repetition System, 간격 반복 시스템)
Lingo-Flow의 핵심 알고리즘입니다. 망각 곡선(Forgetting Curve) 이론에 따라, **기억이 희미해질 시점에 맞춰 복습 주기를 자동으로 배치**합니다.

- **방법**: 앱이 제안하는 '오늘의 복습' 목록을 매일 수행하세요.
- **효과**: 단기 기억을 장기 기억으로 가장 효율적으로 전환합니다.

## 2. Active Recall (능동적 회상)
단순히 단어장을 읽는 것이 아니라, 뇌를 자극하여 정보를 '인출'하는 과정입니다.

- **방법**: 단어의 뜻을 먼저 보지 말고, 단어만 보고 뜻을 떠올리려고 3~5초간 노력한 뒤 정답을 확인하세요.
- **Lingo-Flow 활용**: 퀴즈 모드와 카드 뒤집기 기능을 적극 활용하세요.

## 3. Mnemonics (연상 기법)
이미 알고 있는 이미지, 소리, 이야기와 새로운 정보를 연결하는 기법입니다.

- **방법**:
  - **이미지 연상**: 단어의 의미를 나타내는 강렬한 이미지를 상안하세요.
  - **사운드 연상**: 비슷한 소리가 나는 모국어 단어와 연결하여 재미있는 문장을 만드세요.
  - **장소법 (Method of Loci)**: 익숙한 장소에 단어들을 배치하는 상상을 하세요.

## 4. Contextual learning (맥락 중심 학습)
단어 하나만 외우는 것이 아니라, 그 단어가 쓰이는 '문장'과 '상황'을 함께 기억하는 방식입니다.

- **방법**: Lingo-Flow에서 제공하는 예문을 소리 내어 읽고, 주인공이 된 것처럼 감정을 실어 말해보세요.
- **효과**: 뇌는 의미 있는 연결 고리가 있는 정보를 훨씬 더 잘 기억합니다.

## 5. Daily Log & Multi-sensory (다감각 활용)
보고, 듣고, 쓰고, 말하는 경험을 결합할 때 기억력은 극대화됩니다.

- **방법**: 
  - **말하기**: 단어를 외울 때 반드시 입 밖으로 소리 내어 발음하세요.
  - **인증하기**: 커뮤니티의 'Daily Log' 기능을 통해 오늘 배운 문장을 직접 손으로 써서 사진을 찍거나 타이핑하여 공유하세요.

---

> [!TIP]
> **완벽주의보다는 빈도(Frequency)**가 중요합니다. 하루 1시간 몰아서 공부하는 것보다, 하루 10분씩 3번 나누어 Lingo-Flow를 켜는 것이 뇌 과학적으로 훨씬 유리합니다.
